Salmon Boilerplate

This boilerplate is Justine Win's custom version of Devtip's Starter Kit V2.

Salmon uses Jade and Sass (Gulp) and includes clean flex layout classes, optional bootstrap & skeleton grid, and what-not.

Grids

Decide whether you want to use bootstrap's or skeleton's grid. Go to main.sass and comment out the grid's css file you will not use.

Flex layout

This page is using flex classes. I have included helpful comments on the helper.css file. Go ahead, experiment!

Getting started

Fork or clone, do whatever you want. If you're not into Sass and Jade, you can use Salmon v2 - an HTML5 & CSS3, Non-gulp version.
